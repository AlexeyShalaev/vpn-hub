# Пулы и доступы

Доступ определяет, какие серверы (и какие VPN на них) открыты группе. Настраивается в разделе
**Доступы**.

Доступ группе выдаётся двумя способами, и их можно сочетать:

- **пулом** — целым набором серверов сразу;
- **точечно** — отдельными серверами, вплоть до конкретных VPN на них.

Итоговый (эффективный) доступ группы = серверы её пулов **плюс** точечно выданные серверы.

## Экран «Доступы»

1. Вверху выберите **группу**, для которой настраиваете доступ.
2. Под ней — **сводка**: к скольким серверам группа получает доступ.
3. Ниже два блока: **Пулы серверов** и **Отдельные серверы**.

## Пулы серверов

Пул — это именованный набор серверов (например, «Европа»). Пулы удобны, когда серверов много.

!!! tip "Главное преимущество пула"
    Выдали группе пул — она получила все серверы внутри. Добавили в пул новый сервер — он
    **автоматически** появляется у всех групп, которым этот пул выдан. Не нужно обновлять доступ
    каждой группе вручную.

**Создать пул:** кнопка **«Новый пул»** → задайте название и отметьте серверы в составе.

**Изменить состав:** кнопка **«Состав»** у пула.

**Выдать пул группе:** переключатель напротив пула в блоке «Пулы серверов».

**Удалить пул:** кнопка «Удалить» в окне состава. Группы потеряют доступ, выданный через этот пул.

Когда группа получает сервер через пул, ей открываются **все установленные на нём VPN**.

## Отдельные серверы

Здесь можно выдать доступ точечно и тоньше:

- Переключатель у сервера включает или выключает доступ к нему.
- При включении открываются все установленные на сервере VPN; затем **кнопками-чипами** можно
  оставить только нужные (например, только Amnezia).

Если сервер уже открыт группе **через пул**, у него будет пометка «через пул «…»»: точечно его
переключить нельзя — доступ управляется пулом. Уберите сервер из пула или выключите пул, если
хотите управлять им точечно.

## Как считается итоговый доступ

- **Через пул** → все установленные VPN серверов пула.
- **Точечно** → только выбранные вами VPN, причём из числа реально установленных.
- Итог — объединение того и другого. Один и тот же сервер, выданный и пулом, и точечно, доступен
  группе один раз (со всеми VPN из пула).

## Изменение доступа сразу применяется

Доступ пересчитывается моментально:

- **Расширили** доступ (включили сервер/VPN/пул) — участники группы сразу увидят сервер в разделе
  [Доступно](../member/get-config.md) и смогут получить конфиг.
- **Сузили** доступ (выключили сервер/VPN/пул, удалили пул) — конфиги, к которым участники потеряли
  доступ, **автоматически отзываются** на серверах.

Проверить, кто фактически пользуется конкретным сервером, можно на его странице — см.
[Кто пользуется сервером](clients.md).
