# Устройства

Устройство — это ваш телефон, планшет, компьютер или роутер, на который вы ставите VPN. Конфиги в
VPN Hub выдаются **на конкретное устройство**: так их удобно различать, а при необходимости —
отзывать по отдельности.

Раздел называется **Устройства** (в режиме участника).

## Зачем добавлять устройство

Прежде чем получить конфиг, нужно указать, **куда** вы его ставите. Поэтому порядок такой: сначала
добавьте устройство, потом получайте на него конфиг в разделе [Доступно](get-config.md).

## Добавить устройство

1. Нажмите **«Добавить устройство»**.
2. Укажите **название** — понятное вам, например «Рабочий ноут» или «iPhone».
3. Выберите **платформу**:

    | Платформа | Для чего |
    |---|---|
    | iPhone / iPad | Телефоны и планшеты Apple |
    | Android | Телефоны и планшеты Android |
    | macOS | Компьютеры Mac |
    | Windows | Компьютеры на Windows |
    | Linux | Компьютеры на Linux |
    | Роутер | Домашние роутеры с поддержкой VPN |

4. Нажмите **«Добавить»**.

Платформа влияет на то, какие приложения-клиенты предложит панель при получении конфига.

## Карточка устройства

У каждого устройства показаны **выданные конфиги** — с какого сервера и по какому протоколу. Пока
конфигов нет, будет подсказка получить их на вкладке «Доступно».

## Удаление устройства

Кнопка с корзиной удаляет устройство. Подтвердите:

> «…» будет удалено. Установленные на нём конфиги перестанут отслеживаться.

!!! note "Что удалить надёжнее"
    Удаление устройства убирает его вместе с конфигами из панели. Если ваша задача — прекратить
    доступ конкретного устройства (например, телефон потерян), это правильное действие: связанные
    конфиги снимаются. Само VPN-приложение на устройстве удалите отдельно.
